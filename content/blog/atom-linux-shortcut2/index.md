## Basic commands for editor

**Traversing**

on a line go from word to word

```
ctrl + --> or ctrl + <---
alt + ---> or alt + <---

```

go to the end of the line

```
End
```

go to the beginnig of the line

```
Home
```

go to the top of the file

```
ctrl + Home
```

go to the end of the file

```
ctrl + End
```

**Selection**

select a word

```
ctrl + shift + <---
ctrl + shift + --->
```
